# 💻 AI Code Reviewer

An AI-powered code review tool that analyzes your code, detects issues, suggests improvements, and provides optimized solutions using Gemini AI.

---

## 🚀 Features

- 🔍 Detects bugs and logical errors
- 🧠 Suggests improvements and best practices
- ⚡ Provides optimized and clean code
- 📋 Copy improved code with one click
- 🎨 Modern UI with syntax highlighting
- 📱 Responsive design

---

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

### AI
- Google Gemini API
